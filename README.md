## 🚀 How to Compile and Run

You don't need a heavy IDE to run these files. You can execute any program directly from your terminal/command prompt.

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/AbhirajSinghrajpoot/Java-Program.git](https://github.com/AbhirajSinghrajpoot/Java-Program.git)
   cd Java-Program

```
 2. **Compile the Java file:** (Example: Fibonacci.java)
   ```bash
   javac Fibonacci.java
   
   ```
 3. **Run the compiled class:**
   ```bash
   java Fibonacci
   
   ```
